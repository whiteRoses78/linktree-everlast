# linktree-app

Linktree-Style SaaS-Webapp. Nutzer:innen legen ein Konto an, pflegen eine öffentliche Profilseite mit kuratierten Links.

## Local Development

```bash
npm install
npm run dev
```

Lokale Env-Vars in `.env.local` (siehe `.env.example`).
